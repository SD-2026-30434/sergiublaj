Welcome!

This is the Chef Management System, a full-stack web application developed as part of the Software Design course, 3rd year, Computer Science, Technical University of Cluj-Napoca, 2025–2026.

The application demonstrates key software design principles and patterns including layered architecture, REST API design, state management with NGXS, role-based access control, and component-driven UI development with Angular and PrimeNG.

Features

  • Authentication and authorization with session-based security
  • Role-based access: Admin and Chef roles with distinct permissions
  • CRUD operations for chefs and orders
  • Server-side filtering, sorting, and pagination
  • Reactive state management using NGXS
